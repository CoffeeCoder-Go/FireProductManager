# FireProductManager
É um gerenciador de produto siimples, que não deve ser usado de forma profissional(por enquanto,motivos: falta de testes ee documentação, código completamente desorganizado, falta de segurança e não tem nenhum arquivo de configuração profissional), feito com JavaX Swing no Apache Netbeans,  usando o velho ANT, com banco de dados MySQL JDBC
